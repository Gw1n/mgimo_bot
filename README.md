# МГИМО Профориентация — Telegram-бот

Бот задаёт 3 вопроса с двумя вариантами ответов и подбирает подходящий факультет МГИМО.

## Запуск

1. **Создайте `.env`** файл и вставьте токен бота:
   ```
   BOT_TOKEN=123456:ABC-DEF...
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите бота:**
   ```bash
   python bot.py
   ```

## Структура проекта

| Файл | Описание                                   |
|---|--------------------------------------------|
| `bot.py` | Основная логика бота (хендлеры, запуск)    |
| `content.py` | Все тексты: вопросы, результаты, UI-строки |
| `requirements.txt` | Python-зависимости                         |
| `.env` | Токен бота (постарайтесь не коммитить😉)   |

## Как это работает

- Пользователь отвечает на вопросы бота, после чего по результатам ответов выносится вердикт.
- Ответы на вопросы сохраняются и отображаются в чате.
